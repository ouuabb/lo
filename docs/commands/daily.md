## daily — 创建今日日记

**用法:** `lo daily`

创建包含今日模板的日记文件。模板包含今日完成、待办事项和想法记录三个区块。

### 模板内容

创建的日记文件包含以下模板区块:

- **今日完成** — 记录今天已完成的事项
- **待办事项** — 列出待完成的任务
- **想法记录** — 随手记录灵感和思考

### 文件命名

文件命名格式为 `YYYY-MM-DD-daily.md`，存储在 `resources/` 目录下。

### 示例

```
lo daily
```

### 注意事项

- 同一日期重复运行 `lo daily` 会触发同名冲突，新文件自动入栈（layer>=1）
- 文件创建后需通过 `lo add` + `lo commit` 入库
- 可通过 `lo new "标题" --template daily` 使用模板创建其他日记
