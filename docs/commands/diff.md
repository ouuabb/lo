## diff — 显示文件变更差异

**用法:** `lo diff [路径]`

显示暂存区和未暂存文件的变更差异。

**检测内容:**
- 暂存区新增文件: 显示内容预览
- 暂存区修改文件: 显示旧/新散列、元数据变更（标题、字数）
- 暂存区删除文件: 显示标题和类型
- 未暂存修改文件: 显示散列变更
- 未跟踪文件: 列出文件系统中存在但数据库无记录的文件

加密文件自动解密后对比明文散列。

### 示例

```bash
lo diff                          # 显示所有变更差异
lo diff "resources/笔记.md"      # 查看特定文件的差异
```
