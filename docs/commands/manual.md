## manual — 查看命令手册

**用法:** `lo manual [命令名]`

查看命令的详细参考手册。不带参数时显示所有可用命令的概览和快速索引，指定命令名时显示该命令的详细用法和示例。

### 示例

```
lo manual               # 查看所有命令概览
lo manual init          # 查看 init 命令详情
lo manual auth          # 查看 auth 命令详情
lo manual sync          # 查看 sync 命令详情
```

### 工作机制

- **不带参数**: 显示按功能分组的命令概览（基础命令、版本控制、资源管理、远程同步、知识智能、扩展系统、搜索与查询、安全、其他）
- **带命令名**: 显示该命令的完整信息，包括用法、描述、选项、示例和注意事项

### 注意事项

- `lo manual` 是命令参考手册，侧重用法
- `lo docs` 是功能详解，侧重概念和设计
- `lo help` 是简洁帮助，侧重快速查找

### 相关命令

- [help](help.md) — 简洁帮助
- lo docs — 项目功能详解（架构、加密、认证等概念性内容）
