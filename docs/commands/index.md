## index — 生成索引

**用法:** `lo index`

生成或更新仓库根目录下的 README.md 索引文件。索引内容按类型和时间组织资源列表，包含标题、路径和简要描述。

### 示例

```
lo index
```

### 工作机制

- 从数据库读取所有活跃资源
- 按类型和时间分组排序
- 生成包含标题、路径和简要描述的 Markdown 表格
- 写入仓库根目录的 README.md 文件

### 注意事项

- 每次运行会重新生成整个索引，覆盖已有 README.md
- 仅包含活跃层（layer=0）的资源
